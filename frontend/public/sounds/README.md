# Sound files

Drop your audio files here so `SoundManager` can play them. Paths are built as `/{category}/{soundName}.mp3`.

## Folder structure

```
sounds/
  ui/
    card_drag.mp3
    card_drop.mp3
    draw_card.mp3
    invalid_action.mp3
    turn_start.mp3
  targeting/
    target_select.mp3
    target_confirm.mp3
  cards/
    microwave.mp3
    freeze.mp3
    salt.mp3
    double_salt.mp3
    trade_seats.mp3
    peek.mp3
    foil_wrap.mp3
    shake.mp3
    spoil.mp3
    buffet.mp3
    trash.mp3
    soggy_steam.mp3
  eliminations/
    elimination_heat.mp3
    elimination_freeze.mp3
    elimination_spoil.mp3
    elimination_shake.mp3
    elimination_soggy.mp3
  events/
    round_start.mp3
    final_two_players.mp3
    victory.mp3
```

Missing files are ignored (no console errors). Add files gradually; the game works without them.
